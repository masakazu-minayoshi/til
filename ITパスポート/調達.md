# 調達の流れ  
|利用者側           |  提供者側   |  
|:-----------------|----------:|    
|調達先選定基準の決定 |           |  
|提案依頼書の作成    |           |  
|　　　　　　　　    |提案書の作成 |  
|提案書の評価       |　　　　     |     
|調達先の決定       |           |   
|契約書の締結       | 契約の締結  |    

# 調達の文書  
* **RFP(Request For Proposal)**  
・提案依頼書のこと。  
・利用者がベンダにシステムの提案を依頼するための文書。  
※ベンダー＝販売業者  
・システム化に関する情報共有が目的。  
・システム化の目的、要件、納期などを記載する。  
* **RFI(Reqest For Information)**   
・情報提供依頼書のこと。   
・RFPの提示に先立ってベンダに情報提供を依頼するための文書。  
* RFQ(Reqestt For Quotation)  
・見積依頼書のこと。
