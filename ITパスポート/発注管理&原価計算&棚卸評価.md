# 発注管理  
* **定期発注**：決まった期日に発注を行う方式。  
→需要予測を行い厳密な管理を行う。  
* **定量発注**：決まった量ずつ発注を行う方式。  
→あらかじめ決めた発生時点を下回る発注を行う。  
発注点方式ともいい管理は楽に行える。  
* ２ビン方式：2つの瓶を交互に使う方法。 
# 原価計算  
* 売上原価：売り上げに対する仕入れの原価（費用）のこと。  
・**売上原価 = 機種残高 + 当期仕入 - 期末残高**で求める。

#  棚卸評価  
* **先入先出**：先に入った古いものを先に出す。  
* 後入先出：後に入った新しいものを先に出す。
