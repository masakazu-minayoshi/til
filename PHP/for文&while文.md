# for文（繰り返す数が決まっている）  
```
<?php
for($i = 0; $i < 10; $i++){
  // 繰り返しを行うの中での条件
  if($i === 5) {
  //処理が止まる  
  break;
  }
  //処理の内容
  echo $i;
}
▶️出力結果：01234

for($i = 0; $i < 10; $i++){
  // 繰り返しを行うの中での条件
  if($i === 5) {
  //条件にした内容をスキップする
  continue;
  }
  //処理の内容
  echo $i;
▶️出力結果：012346789
}

# while文(繰り返す数が決まっていない)
$i = 0;
while($j < 5) {
  echo $j;
  $j++;
}
▶️出力結果：1234
?>
```
