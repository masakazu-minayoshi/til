# 繰り返し処理  
## while構文  
```  
初期化処理
While(繰り返す条件) {
  繰り返したい処理　
  更新処理
  }
  ```
（例）1~365の数字を表示させる。(sample06.php)    
  ```
<?php
$i = 1;
while($i <= 365) {
  print ($i."\n");
  $i ++; //インクリメント（加算）で行った場合
}
?>
  ```
（例）1~365の数字を表示させる。(sample06.php)    
  ```
 ①【繰り返しの終わり位置がわかりやすい】
<?php
$i=1;
while($i <=365):
  $date = strtotime('+' .$i. 'day');
  print(date('n/j(D)', $date));
  print "\n";
  $i++;
endwhile;
?>
②
<?php
$i=1;
while($i <=365) {
  $date = strtotime('+' .$i. 'day');
  print(date('n/j(D)', $date));
  print "\n";
  $i++;
}
?>
  ```
## for構文  
```  
for (初期化処理; 繰り返す条件; 更新処理) {
   繰り返したい処理
 }
  ```
  （例）1~365の数字を表示させる。(sample06.php)    
  ```
<?php
for ($i = 1; $i <= 365; $i++) {
  print ($i."\n");
}
?>
  ```
（例） 1年後までのカレンダーを表示させる。    
  ```
①【繰り返しの終わり位置がわかりやすい】
<?php
for ($i=1; $i<=365; $i++):
  $date = strtotime('+' .$i. 'day');
  print(date('n/j(D)', $date));
  print "\n";
endfor;
?>
②
<?php
for ($i=1; $i<=365; $i++) {
  $date = strtotime('+' .$i. 'day');
  print(date('n/j(D)', $date));
  print "\n";
}
?>
※strtotime：string to timeの略。文字列で表す。
  ```
