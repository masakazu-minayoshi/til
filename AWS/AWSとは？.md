# AWSとは何？  
* AWS(Amazon Web Services)はAmazon社が提供する**クラウドサービス**。  
* 100以上のサービスが存在し、クラウドサービスとして世界最大規模である。  
## AWSの特徴  
①サービスが豊富  
* EC2やRDSなど100以上のサービスが存在している。  
* 高負荷に耐えられる、信頼性の高いシステムを少ない手間で運用できる。  
②リソースが柔軟  
* リソースを必要な時に必要な分だけ調達できる。  
（例）繁忙期の時だけサーバーを増やす。  
③従量課金  
* 使った分だけ支払う従量課金なので費用対効果が優れている。
