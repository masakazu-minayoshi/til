# 変数の定義  
## int型の変数定義
```
//データ型 変数名
int number;
//代入
number = 3;
//出力
System.out.println(number);
>出力
10
```
## String型の変数定義
```
//データ型 変数名
String name;
//代入
number = Sato;
//出力
System.out.println(name);
>出力
Sato
```
