# 真偽値  
## データ型  
* 真偽値のデータは**boolean型**。

|データ型|表示|
|:---:|:---:|
|boolean型|true<br>flase|
|String型|"true"<br>"flase"|

## boolean型の変数定義
```
//データ型、変数名、値
boolean bool = true;
```

# 比較演算子  

|内容|詳細|
|:---:|:---:|
|x == y|xとyが等しい時にtrue<br>xとyが等しくない時にflase|
|x != y|xとyが等しくない時にtrue<br>xとyが等しい時にflase|
